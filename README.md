# sbt-console-template

Get started with Scala and use Intellij IDEA (tested on 11)

* Scala 2.9.1
* ScalaTest 1.6.1

## Using

Run the app

    sbt run

Run tests

    sbt test


## Convert to IDEA project

1. Start sbt

    sbt

2. Within sbt, generate IDEA project

    > gen-idea
    > exit

3. Open the project in IDEAj from the menu, File -> Open Project ...
